# Block.One-BlockChain-Viewer


### Requirements
* Swift 4.0
* iOS 8.0+
* Xcode 9.0

*iOS Application to view the most recent blocks on the EOS Blockchain.
The testing of this API was done on the following endpoint: https://node1.eosphere.io/v1 *
